# AI Website Chatbot with n8n & Google Gemini

An AI-powered chatbot workflow that answers user questions based on website content.

## Features

- Real-time website content retrieval
- HTML content extraction
- AI-powered question answering
- Context-aware responses
- Memory support
- No-code/low-code implementation

## Workflow

User Question
    ↓
HTTP Request
    ↓
HTML Extraction
    ↓
Google Gemini AI Agent
    ↓
Answer Generation

## Tech Stack

- n8n
- Google Gemini
- HTTP Request Node
- HTML Extract Node
- AI Agent
- Simple Memory

## Use Cases

- Customer Support Automation
- FAQ Assistants
- Business Websites
- Educational Platforms
- Knowledge Base Search
